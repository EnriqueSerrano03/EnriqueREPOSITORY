Instalación desde terminal

-Para instalar desde la terminal hay que poner una serie de comando los cuales los he resumido en uno solo.

- sudo apt-get install update && sudo apt-get install clamav clamav-daemon

-Si deseamos ver la versión que hemos installado ponemos el siguiente comando ☞ clamscan --version

-Esta versión solo se puede usar mediante comandos

Instalar versión grafica
